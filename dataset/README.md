# Dataset

This project uses the **Twitter Sentiment Dataset (3 Million Labelled Rows)** available on Kaggle.

## Source

https://www.kaggle.com/datasets/prkhrawsthi/twitter-sentiment-dataset-3-million-labelled-rows

## Description

The dataset contains millions of labelled Twitter posts for sentiment analysis and machine learning research.

## Dataset Access

The dataset is **not included** in this repository due to its large size and to respect the original distribution source.

Please download it directly from Kaggle using the link above.
